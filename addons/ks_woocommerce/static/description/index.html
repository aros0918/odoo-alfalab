Connect Multiple WooCommerce Stores

With a single Odoo database, you can connect multiple WooCommerce stores.

Import Operations

Import the following from WooCommerce to Odoo via three manual ways (Export All, Specific Id Filter, or Date Filter) add 'or automatically through webhooks'

Export Operations

Export the following from Odoo to WooCommerce via three manual ways (Export All, Specific Id Filter, or Date Filter)

Order Lifecycle

Import all orders (shipped/unshipped) from WooCommerce to Odoo. Update order status from the Odoo to WooCommerce.

Bidirectional Product Syncing

Import and export all products from WooCommerce to Odoo or Odoo to WooCommerce.

Bidirectional Stock Level Syncing

Automatically update the product stock level with bidirectional sync between Odoo and WooCommerce.

Logs of Failed Jobs

With Failed Logs, a user can track the status of failed import/export functionality.

Informative Dashboard

The dashboard represents the New State, Failed State, Progress State, and Completed State of all the domains using a Bar Chart or a Line Chart. A user can also check the counts of the successfully synched domains.

Sales Reporting

Sales Reporting allows a user to create a sales report of a particular WooCommerce instance.

Publish and Unpublish product in Woocommerce from Odoo

A user can publish or unpublish a product in a WooCommerce store through the Odoo backend. This will save store owners enough time to manually add/remove products at WooCommerce store.

Data Syncing

Synchronize all the data from WooCommerce to Odoo or Odoo to WooCommerce with three manual options:

Import/Export All Special ID Filter Date Filter
Manage Payment Methods

Easily import all the payment gateways related to orders while importing orders from WooCommerce to Odoo.

Meta Mapping

With the Meta Mapping feature, a user can map the WooCommerce custom fields with the Odoo fields while importing/exporting any data.

Manage Refunds

Whenever there is a need to refund an order, a user can initiae refunds from Odoo.

Automatic Workflow for Order Import

After importing an order, a user can automate the few functionalities of an order in Odoo from the given Workflow Options: Validate Order, Create Invoice, Register Payment, Force Invoice Date, and Confirm Shipment.

Daily Sales Report via Email

A user can send the daily sales report to multiple email addresses on an hourly or monthly basis.

Webhook Logging

With the Webhook logs feature, a user can quickly check the records of all the webhooks performed at the WooCommerce end in Odoo.

Set Automatic Jobs via Webhooks

For any Order, Product, Customer, & Coupon updated/created at the WooCommerce end it can automatically sync in the Odoo through webhook actions.
