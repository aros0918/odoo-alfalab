Product Kit

This Application consists of a fixed set of components that are sold as a unit. A product kit is not a stocked inventory item; however, its components may be. The fixed set of components are not required to ship together unless specified in the kit definition.

This module is compatible with

  • Community
  • Enterprise
  • Odoo.sh

Key Features

    Allows to configure product as a group/kit of products.
    In Sales order use kit product and create invoice only for the main kit product.
    Manages stock for sub-products of a kit.
    Allow to Return a kit with sub-products.
    Bifurcation of all sub-products separately in (Sale, Invoice) Reports which are used in a kit.
    Can be also used for eCommerce Shop.

Creating Product Kit Using the Product Definition

Path For Product Create: Sales 🠦 Products 🠦 Products

Select the Product Kit option on the Product Definition - Definition page (Products, Identify Product Details, Definition) to identify the product ID as the parent/kit product. Kit sub products option fields appear.

Assigning Components to Product Kit

Path For Product Create: Sales 🠦 Products 🠦 Products

Use the Sub products Summary page to define the components of a product kit.

Product: You can enter both stocked and nonstock products in the field as long as the nonstock product is not a product kit. That is, you cannot attach a product kit as a component of another product kit.

Quantity: Define the quantity that the component can have in Kit.

'Is Product Kit' will Filter out all Products defined as Kit.

Use kits in sales order

Path For Sale Order Create: Sales 🠦 Orders 🠦 Quotations

User can add kit products in sale order lines.

Once sales order is Confirmed then Delivery is creating with all sub-products.

Path For Return Transfers: Sales 🠦 Orders 🠦 Delivery(Button) 🠦 Retrun(Button)

Return all kit sub products, not allowed to return partial kit.

Return Delivery with all sub products.

Path For S/O Invoice: Sales 🠦 Orders 🠦 Orders 🠦 Create Invoice(Button)

Invoice is created for the main kit product, not for the sub products.

Reports

Path For Sale Order Report: Sales 🠦 Reporting 🠦 Sales

Easy to check sales by Product Kit.

Path For S/O Report: Sales 🠦 Quotations 🠦 Print 🠦 Quotations / Order

While Generating a Sales Report Product Kit will be bifurcated with all of their included Sub Product and will be displayed on the Report.

Path For S/O Invoice Report: Invoicing 🠦 Invoices 🠦 Print 🠦 Invoices

While Generating a Invoice Report Product Kit will be bifurcated with all of their included Sub Product and will be displayed on the Report.
